¿Que es Redis?

Redis es un sistema de gestión de bases de datos en memoria, de código abierto y con estructura de datos clave-valor. Es conocido por su alto rendimiento, flexibilidad y soporte para diversas estructuras de datos como cadenas, listas, conjuntos y hashes. Redis se utiliza comúnmente como base de datos, caché y broker de mensajes, y es especialmente popular en aplicaciones que requieren baja latencia y alta disponibilidad.

Caracteristicas de Redis

Almacenamiento en memoria

  • Organiza la información en estructuras de datos clave-valor.
  • Permite un acceso extremadamente rápido a los datos almacenados.

Modelo clave valor avanzada

  • Permite la manipulación de datos en tiempo real con baja latencia.
  • No solo maneja cadenas simples, también soporta estructuras como: listas, conjuntos, hashes, conjuntos ordenados, hiperbólicos, mapas de bits, streams y geodatos.

Persistencia opcional

  • Ofrece opciones para guardar datos en disco, permitiendo la recuperación después de un reinicio.
  • RDB (Redis Database Backup): crea snapshots periódicos.
  • AOF (Append Only File): guarda cada operación para recuperarlas después.

Altísimo rendimiento

  • Ofrece latencias extremadamente bajas, lo que lo hace ideal para aplicaciones en tiempo real.
  • Puede manejar millones de operaciones por segundo en hardware estándar.

Alta disponibilidad y recuperación

  • Herramientas como Redis Sentinel y Redis Cluster permiten tolerancia a fallos y respaldo confiable.

Soporte para replicación y alta disponibilidad

  • Replica datos en tiempo real a través de múltiples nodos.
  • Permite la conmutación por error automática en caso de fallos.

Escalabilidad

  • Permite escalar horizontalmente añadiendo más nodos al clúster.

Transacciones

  • Permite realizar múltiples operaciones de forma atómica.
  • Garantiza la consistencia de los datos en entornos concurrentes.

Publicación y suscripción (Pub/Sub)

  • Permite la gestión de mensajes en tiempo real entre productores y consumidores.
  • Facilita la creación de aplicaciones reactivas y basadas en eventos.

Ligero y de código abierto

  • Es open source y tiene una instalación muy sencilla.
