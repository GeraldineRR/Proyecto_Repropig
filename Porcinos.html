Diccionario de datos base de REPROPIG
Nombre Porcinos
Creación 06/08/2025
Descripción Registro de datos propios para los lechones de la unidad de porcinos en el centro agropecuario "La Granja"
Campo Tipo de dato Tamaño Descripción
Id INT 10 Número identificador para cada cerdo
Id_chapeta INT 10 Representa el número de chapeta con el que cada porcino es identificado
Nombre VARCHAR 24 Nombre con el cual se reconoce al porcino
Fecha de nacimiento DATE 0 Muestra la fecha exacta en la que ha nacido el porcino
Genero ENUM ('F','M') Define el genero al cual pertenece el lechon
Raza VARCHAR 50 Define el tipo de raza a la que pertenece el lechon
Id_Madre INT 11 Identificador con el cual es reconocida la madre del porcino mencionado
Id_Padre INT 11 Identificador con el cual es reconocido el padre del porcino mencionado
