Descripcion Registro de datos de los nacimientos de la unidad de porcinos del centro Agropecuario La Granja
Campo Tipo dato Tamaño Descripcion
Id_Nacimientos INT 11 Indicador del nacimiento registrado
Id_parto INT 11 Idenficación del parto al cual pertenecen los nacimientos
Nacidos vivos INT 2 Representa el numero de porcinos nacidos vivos
Nacidos muertos INT 2 Representa el Numero de porcinos nacidos muertos
Nacidos momias INT 2 Representa el Numero de porcinos nacidos momias
Sacrificados INT 2 Representa el Numero de porcinos sacrificados
Aplastados INT 2 Representa el Numero de porcinos aplastados por la madre
cantidad Hembras INT 2 Representa la cantidad de Hembras
cantidad Machos INT 2 Representa la cantidad de Machos
