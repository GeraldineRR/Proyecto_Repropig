PostgreSQL

PostgreSQL es una base de datos de objetos-relacional de código abierto, avanzada y de clase empresarial, conocida por su robustez, integridad y fiabilidad. Es compatible con SQL y soporta la gestión de datos JSON, y ofrece características como transacciones ACID, tipos de datos complejos y funciones avanzadas.

Características Clave

  • Código Abierto: gratuita y con gran comunidad de desarrolladores.
  • Relacional de Objetos: maneja objetos y datos relacionales.
  • Soporte ACID: garantiza la integridad de los datos.
  • Compatibilidad: soporta SQL y JSON.
  • Extensibilidad: permite definir tipos y funciones propias.
  • Seguridad: claves foráneas, vistas y control de acceso.
  • Alta Disponibilidad: soporta múltiples conexiones.
  • Multiplataforma: funciona en Linux, Windows y macOS.

Sentencias SQL CRUD

1. Crear (INSERT)

INSERT INTO nombre_tabla (columna1, columna2, ...)
VALUES (valor1, valor2, ...);

2. Consultar (SELECT)

SELECT columna1, columna2, ...
FROM nombre_tabla
WHERE condicion;

3. Actualizar (UPDATE)

UPDATE nombre_tabla
SET columna1 = valor_nuevo1, columna2 = valor_nuevo2, ...
WHERE condicion;

4. Eliminar (DELETE)

DELETE FROM nombre_tabla
WHERE condicion;
