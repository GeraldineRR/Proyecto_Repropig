Características de MySQL

  • Código abierto y gratuito: MySQL es software libre bajo la licencia GNU GPL, aunque también tiene versiones comerciales ofrecidas por Oracle.
  • Basado en SQL: Utiliza el lenguaje estándar SQL (Structured Query Language) para gestionar y manipular bases de datos.
  • Multiplataforma: Funciona en diversos sistemas operativos como Linux, Windows, macOS, y más.
  • Alta velocidad y rendimiento: Optimizado para ofrecer un buen rendimiento, especialmente en consultas de lectura.
  • Escalabilidad: Puede manejar desde pequeñas bases de datos personales hasta grandes sistemas con millones de registros.
  • Soporte para múltiples motores de almacenamiento:
    • InnoDB (por defecto, con soporte para transacciones y claves foráneas).
    • MyISAM (más rápido para lecturas, pero sin soporte de transacciones).
    • Otros: MEMORY, CSV, ARCHIVE, etc.
  • Seguridad y control de acceso: Soporta autenticación de usuarios, privilegios por tabla/columna, y cifrado de conexiones (SSL).
  • Transacciones: Compatible con transacciones ACID (si se usa InnoDB), lo que permite mayor integridad de los datos.
  • Replicación: Soporta replicación maestro-esclavo y maestro-maestro, útil para balanceo de carga y respaldo.
  • Integración con múltiples lenguajes: Compatible con muchos lenguajes de programación: PHP, Python, Java, C#, Perl, etc.
  • Interfaces gráficas y herramientas: Herramientas como MySQL Workbench, phpMyAdmin, entre otras, facilitan la administración.
  • Comunidad y documentación: Amplia comunidad de usuarios y una excelente documentación oficial.

Operaciones básicas en MySQL

Operación Descripción Sentencia MySQL
CREATE Inserta un documento en la colección INSERT INTO nombre_tabla (columna1, columna2, ...)VALUES (valor1, valor2, ...);
READ Busca documentos que coincidan con la condición SELECT columna1, columna2 FROM nombre_tabla WHERE condición;
UPDATE Actualiza un documento que cumple el filtro UPDATE nombre_tabla SET columna1 = valor1, columna2 = valor2 WHERE condición;
DELETE Elimina un documento que cumple la condición DELETE FROM nombre_tabla WHERE condición;
