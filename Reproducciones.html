Creacion 27/03/2025
Descripcion Registro de datos reproductivos para las cerdas del centro agropecuario la granja
Campo Tipo dato Tamaño Descripcion
Id_Reprodcuciones INT 11 Numero dentificador para los datos reproductivos
Id_Porcino INT 11 Atrae los datos de la cerda a reproducir
MachoRep_Id_Porcino INT 11 Atrae los datos del macho reproductor
Fecha_Servicio DATE 0 Es la fecha en que la cerda fue inseminada artificialmente o montada por un macho
Tipo_Servicio ENUM ('Monta','Inseminación') Es la fecha en que la cerda fue inseminada artificialmente o montada por un macho
FPP DATE 0 Es la fecha estimada en que la cerda debe parir si la gestación va bien.
Resultado ENUM ('Pendiente','1RC','2RC','Preñada','Vacia','Aborto') Indica el resultado que se obtuvo al hacer el proceso reproductivo
Observaciones VARCHAR 50 Campo opcional para reportar observaciones del proceso de reproducción
